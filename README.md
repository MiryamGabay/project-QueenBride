## Queen Bride

 - **The project's description.**

 Queue management system for bridal salons.
 Through the system you can book an appointment, manage the queues and the type of          service and also manage the service providers.

 - **Entities :** 
                       1. Customer -bride.
                       2.Turn.
                       3.service provider
